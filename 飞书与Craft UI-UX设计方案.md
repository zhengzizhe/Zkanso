## 1. 页面布局结构

### 1.1 飞书文档布局架构

#### 主界面区域划分

**顶部导航栏（固定）**
- 位置：屏幕顶部，固定不滚动
- 高度：48-56px
- 包含元素：
  - 左侧：产品 Logo、返回按钮
  - 中间：文档标题/搜索框
  - 右侧：分享按钮、用户头像、更多操作（三点图标）

**左侧边栏（可收起）**
- 宽度：240-280px（展开状态），48px（收起状态）
- 导航层级：
  - 主页
  - 我的空间
  - 共享空间
  - 知识库
  - 百科
  - 收藏
  - 回收站
- 交互特性：鼠标悬停显示完整菜单，支持快捷键折叠（Ctrl/Cmd + \）

**主内容区**
- 宽度：动态自适应，最大内容宽度 800-1000px（居中显示）
- 包含：
  - 文档标题区（大号字体）
  - 工具栏区（浮动工具栏）
  - 正文编辑区
  - 目录导航（右侧浮动）

**右侧辅助栏（可选）**
- 宽度：240-300px
- 包含：
  - 大纲/目录（自动生成）
  - 评论面板
  - 历史版本
  - 文档信息

#### 响应式布局规则

**桌面端（>1280px）**
- 三栏布局：左侧导航 + 中间内容 + 右侧辅助
- 内容区最大宽度限制，确保阅读舒适度

**平板端（768px - 1280px）**
- 两栏布局：左侧导航（可收起）+ 内容区
- 右侧辅助栏通过按钮调出（浮层形式）

**移动端（<768px）**
- 单栏布局
- 导航通过汉堡菜单（抽屉式）调出
- 工具栏简化为浮动按钮组
- 底部固定操作栏（编辑、评论、分享）

### 1.2 Craft 布局架构

#### 主界面区域划分

**双侧边栏系统（Web/iPad）**
- 左侧边栏 1（组织面板）：
  - 宽度：220-260px
  - 内容：空间（Spaces）、文件夹、文档树
  - 支持拖拽排序和嵌套
  
- 左侧边栏 2（大纲/日历）：
  - 宽度：200-240px
  - 内容：文档大纲（Table of Contents）或日历视图
  - 可通过滑动手势快速切换

**主编辑区**
- 宽度：自适应，最大 720px（优化阅读）
- 特点：
  - 极简设计，无固定工具栏
  - 浮动工具栏（选中内容时出现）
  - 块级编辑系统（Block-based）
  - 支持卡片式内容组织

**顶部操作栏（轻量级）**
- 高度：40-48px
- 包含：
  - 左侧：导航返回、文档标题
  - 右侧：分享、更多选项、主题切换

#### 响应式适配

**Mac/iPad（大屏）**
- 双侧边栏 + 主编辑区
- 支持多窗口、标签页管理
- 全局搜索浮层（Cmd + K）

**iPhone（移动端）**
- 下拉导航面板（Pull-down Navigation）
- 单栏编辑模式
- 浮动工具栏位于键盘上方
- 手势支持：滑动返回、长按预览

**布局比例**
- 左侧导航占比：15-20%
- 主内容区占比：60-70%
- 右侧辅助占比：15-20%（如有）

## 2. 界面元素设计

### 2.1 飞书设计规范

#### 色彩系统

**主色调**
- 品牌蓝：#3370FF（Primary Blue）
- 应用场景：链接、按钮、选中状态、重要提示

**辅助色**
- 成功绿：#00B578
- 警告橙：#FF8F1F
- 错误红：#F54A45
- 信息蓝：#3370FF

**中性色（灰度系统）**
- 文本主色：#1F2329（正文）
- 文本次色：#646A73（辅助文字）
- 文本禁用：#C9CDD4
- 边框色：#DEE0E3
- 分割线：#F2F3F5
- 背景色：#FFFFFF, #F7F8FA, #EBEDF0

**功能色映射**
- 悬停态：主色 + 10% 透明度
- 激活态：主色加深 8%
- 禁用态：灰度 #C9CDD4

#### 文字排版体系

**字体家族**
- 中文：PingFang SC（苹果系统）、Microsoft YaHei（Windows）
- 英文/数字：Roboto、Helvetica Neue
- 等宽字体（代码）：Monaco、Consolas

**字号层级**
| 层级 | 字号 | 行高 | 应用场景 |
|------|------|------|----------|
| H1 | 24px | 32px | 一级标题 |
| H2 | 20px | 28px | 二级标题 |
| H3 | 16px | 24px | 三级标题 |
| Body Large | 16px | 24px | 重要正文 |
| Body | 14px | 22px | 常规正文 |
| Body Small | 13px | 20px | 辅助文字 |
| Caption | 12px | 18px | 说明文字 |

**字重规范**
- Regular (400)：常规文本
- Medium (500)：强调文本
- Semibold (600)：小标题
- Bold (700)：重要标题

**段落间距**
- 标题与正文：12-16px
- 段落之间：8-12px
- 列表项间距：4-8px

#### UI 组件库

**按钮设计**
- 主按钮（Primary）：
  - 背景：#3370FF
  - 文字：#FFFFFF
  - 圆角：4px
  - 高度：32px / 36px / 40px
  - 内边距：12px-20px
  - 悬停：背景加深 8%

- 次要按钮（Secondary）：
  - 边框：1px solid #DEE0E3
  - 背景：#FFFFFF
  - 文字：#1F2329
  - 悬停：边框变为品牌蓝

- 文本按钮（Text）：
  - 无边框、无背景
  - 文字颜色：#3370FF
  - 悬停：背景 #F2F3F5

**输入框**
- 默认状态：
  - 边框：1px solid #DEE0E3
  - 圆角：4px
  - 高度：32px
  - 内边距：8px 12px
  - 占位符颜色：#C9CDD4

- 焦点状态：
  - 边框：1px solid #3370FF
  - 外阴影：0 0 0 3px rgba(51, 112, 255, 0.1)

- 错误状态：
  - 边框：1px solid #F54A45
  - 错误提示文字：12px，#F54A45

**卡片 / 容器**
- 背景：#FFFFFF
- 边框：1px solid #DEE0E3（可选）
- 圆角：8px
- 阴影：0 2px 8px rgba(0, 0, 0, 0.08)
- 内边距：16px / 20px / 24px

**图标规范**
- 尺寸：16px、20px、24px、32px
- 线条宽度：1.5px - 2px
- 风格：线性图标为主
- 颜色：继承文本颜色或独立设置

### 2.2 Craft 设计规范

#### 色彩系统（极简美学）

**核心配色**
- 默认背景：#FFFFFF（纯白）
- 文本主色：#000000（纯黑）
- 文本次色：#6B6B6B（中灰）
- 边框/分割线：#E5E5E5（浅灰）

**品牌色（最小化使用）**
- 强调色：#FF6B6B（红橙，用于重要提示）
- 链接色：#0066CC（蓝色）
- 选中色：#D4E8FF（浅蓝背景）

**主题系统**
- 100+ 预设主题样式
- 支持自定义：
  - 纸张颜色（Paper Color）
  - 文本颜色（Text Color）
  - 背景图片（Unsplash 集成）
  - 渐变背景（Gradient）

**深色模式**
- 背景：#1C1C1E
- 文本：#FFFFFF
- 次要文本：#A0A0A0
- 卡片：#2C2C2E

#### 文字排版体系

**字体家族**
- 系统默认：San Francisco (iOS/Mac)、Segoe UI (Windows)
- 可选字体：
  - 衬线：New York、Georgia、Playfair Display
  - 无衬线：Inter、Helvetica、Avenir
  - 等宽：SF Mono、JetBrains Mono

**字号层级**
| 层级 | 字号 | 行高 | 应用场景 |
|------|------|------|----------|
| Title | 28px | 36px | 文档标题 |
| Heading 1 | 24px | 32px | 一级标题 |
| Heading 2 | 20px | 28px | 二级标题 |
| Heading 3 | 18px | 26px | 三级标题 |
| Body | 17px | 26px | 正文（iOS推荐） |
| Small | 15px | 22px | 辅助文字 |
| Caption | 13px | 18px | 说明文字 |

**排版特点**
- 行高比例：1.5 - 1.8 倍
- 字间距：正常（0）或略微增加（0.02em）
- 段落间距：1.2 - 1.5 倍行高
- 对齐方式：左对齐为主

#### UI 组件设计

**块（Block）系统**
- 文本块：
  - 无边框
  - 悬停显示拖拽手柄（左侧）
  - 点击空行显示"+"按钮（插入新块）

- 卡片块：
  - 背景：略带阴影的白色卡片
  - 圆角：12px
  - 阴影：0 2px 12px rgba(0, 0, 0, 0.06)
  - 内边距：20px
  - 支持嵌套子块

- 媒体块（图片/视频）：
  - 最大宽度：100%
  - 圆角：8px
  - 悬停显示操作菜单

**浮动工具栏**
- 位置：选中内容上方/键盘上方（移动端）
- 背景：#FFFFFF（带阴影）
- 圆角：8px
- 高度：40-44px
- 包含功能：
  - 文字格式：粗体、斜体、下划线、删除线
  - 颜色：文本颜色、背景色
  - 链接插入
  - 块类型切换（标题、列表、引用等）

**装饰元素**
- Washi Tape（和纸胶带）：
  - 作为分隔符
  - 手绘风格
  - 多种颜色和图案
  
- Doodles（涂鸦分割线）：
  - 手绘风格的分隔线
  - 增添趣味性

**按钮/交互元素**
- 极简设计：无边框、无阴影
- 悬停状态：背景 #F5F5F5
- 激活状态：背景 #E5E5E5
- 圆角：6px

## 3. 功能模块交互逻辑

### 3.1 飞书文档交互设计

#### 导航结构

**顶部导航**
- 面包屑导航：
  - 显示当前文档路径
  - 可点击快速跳转
  - 箭头分隔符（>）
  
**侧边菜单**
- 层级展开/收起：
  - 点击箭头展开子目录
  - 最多支持 5 级嵌套
  - 当前选中项高亮（背景 #F2F3F5）
  
- 搜索功能：
  - 全局搜索：Ctrl/Cmd + K
  - 实时搜索结果
  - 高亮匹配文本
  - 支持拼音首字母搜索

**右键菜单**
- 文档操作：
  - 新建文档
  - 复制链接
  - 移动到
  - 删除
  - 权限设置

#### 编辑器功能

**浮动工具栏（Floating Toolbar）**
- 触发方式：
  - 选中文本时自动出现
  - 位置：选中区域上方（避免遮挡）
  - 延迟：150ms（避免误触发）

- 工具分组：
  - 基础格式：粗体、斜体、下划线、删除线、代码
  - 颜色：文字颜色、背景色（色板）
  - 链接：插入/编辑链接
  - 评论：添加批注

**斜杠命令（/Command Menu）**
- 触发：输入"/"
- 功能：
  - 快速插入块类型
  - 分类：
    - 基础块：文本、标题、列表
    - 媒体：图片、视频、音频
    - 嵌入：代码块、表格、Mermaid图表
    - 高级：投票、日程、分栏
  - 搜索过滤：输入关键词快速定位

**区块菜单（Block Menu）**
- 触发方式：
  - 鼠标悬停在行首显示"⋮⋮"图标
  - 点击展开菜单
  
- 菜单选项：
  - 拖拽移动（上下调整顺序）
  - 转换块类型
  - 复制/删除
  - 添加评论
  - 生成链接

**Markdown 支持**
- 实时转换：
  - `# ` → 一级标题
  - `## ` → 二级标题
  - `- ` 或 `* ` → 无序列表
  - `1. ` → 有序列表
  - `` `code` `` → 行内代码
  - `[]()` → 链接
  - `> ` → 引用

#### 模态框与弹窗

**分享弹窗**
- 尺寸：480px × 600px（居中）
- 背景遮罩：rgba(0, 0, 0, 0.4)
- 内容：
  - 权限设置（可查看/可编辑/禁止访问）
  - 链接生成
  - 密码保护
  - 有效期设置
  - 成员列表

**图片预览**
- 全屏浮层
- 背景：rgba(0, 0, 0, 0.9)
- 操作：
  - 关闭按钮（右上角）
  - 缩放控制（滚轮）
  - 旋转、下载按钮

**确认对话框**
- 尺寸：400px × 200px
- 结构：
  - 图标（警告/提示）
  - 标题
  - 描述文字
  - 按钮组（取消 + 确认）

**Toast 提示**
- 位置：顶部居中（距顶部 24px）
- 持续时间：2-5 秒
- 类型：
  - 成功：绿色图标
  - 错误：红色图标
  - 警告：橙色图标
  - 信息：蓝色图标

### 3.2 Craft 交互设计

#### 导航交互

**双侧边栏切换**
- Mac/iPad：
  - 滑动手势切换"组织面板"和"大纲面板"
  - 快捷键：Cmd + 1（组织）、Cmd + 2（大纲）
  
- iPhone：
  - 下拉面板（Pull-down Navigation）
  - 从屏幕顶部下滑显示空间导航
  - 点击空白处或上滑关闭

**多窗口/标签页**
- Mac 支持：
  - 多窗口并列显示
  - 标签页管理（类浏览器）
  - 拖拽标签生成新窗口

**全局搜索（Cmd + K）**
- 浮层样式：
  - 居中弹出（600px × 400px）
  - 模糊背景（backdrop-filter: blur(10px)）
- 功能：
  - 实时搜索文档
  - 快速操作（Quick Actions）
  - 最近访问记录

#### 编辑器交互

**块级编辑（Block-based）**
- 点击空行：
  - 显示"+"按钮
  - 展开块类型选择器
  - 支持键盘上下键选择

- 块拖拽：
  - 悬停显示拖拽手柄（左侧 6 点图标）
  - 拖拽时显示蓝色插入线
  - 支持跨文档拖拽

- 块嵌套：
  - Tab 键增加缩进
  - Shift + Tab 减少缩进
  - 最多 6 级嵌套

**浮动工具栏（移动端）**
- 位置：键盘上方
- 高度：44px
- 包含：
  - 快速格式按钮（B、I、U）
  - 颜色选择器（Color Picker）
  - 标题、列表切换
  - 链接插入

**代码编辑器**
- 特点：
  - 20+ 主题
  - 语法高亮
  - 行号显示
  - 紧凑 UI
  - 一键插入即可编辑
  - 三点浮动菜单（快速选项）

**白板功能（Whiteboard）**
- 插入方式：
  - "/"命令或"+"按钮
- 交互：
  - 自由绘制
  - 拖拽元素
  - 协作光标（多人）

#### 手势与快捷键

**拖拽交互**
- 文档拖拽：
  - 空间间移动文档
  - 拖拽到文件夹（嵌套）
  - 拖拽到其他应用分享

- 块拖拽：
  - 长按激活拖拽模式（移动端）
  - 拖拽重新排序
  - 拖拽到侧边栏创建链接

**手势支持（iOS/iPadOS）**
- 双指捏合：缩放（白板）
- 双指滑动：前进/后退导航
- 长按预览：轻量级预览文档内容

**键盘快捷键（macOS）**
- 文档管理：
  - Cmd + N：新建文档
  - Cmd + F：文档内搜索
  - Cmd + K：全局搜索
  
- 编辑：
  - Cmd + B：粗体
  - Cmd + I：斜体
  - Cmd + K：插入链接
  - Cmd + /：显示所有快捷键
  
- 样式：
  - Ctrl + 1-6：切换标题级别
  - Cmd + Shift + T：任务列表
  
- 导航：
  - Cmd + ← / →：前进/后退
  - Cmd + .：聚焦模式

## 4. 用户体验细节

### 4.1 飞书体验亮点

#### 视觉层次构建

**22 个优秀体验细节**

1. **新建知识空间预览**
   - 鼠标悬停触发下拉面板
   - 模板预览（缩略图）
   - 减少点击次数

2. **模板联动推荐**
   - 输入标题自动推荐相关模板
   - 智能匹配场景
   - 提升使用便捷性

3. **浮动工具栏设计**
   - 选中内容时出现
   - 工具跟随光标位置
   - 避免固定工具栏占用空间

4. **Markdown 自动识别**
   - 无需手动切换模式
   - 提升输入效率
   - 专注内容而非格式

5. **权限管理清晰**
   - 多人协作实时同步
   - 权限分级明确
   - 操作历史可追溯

6. **模板库丰富**
   - 覆盖多种工作场景
   - 专业且实用
   - 所有模板免费

7. **分享功能灵活**
   - 多种权限设置
   - 支持密码保护
   - 有效期控制

8. **实时协作**
   - 多人光标显示
   - 修改即时同步
   - 冲突自动合并

#### 交互流程优化

**效率指南**
- 位置：文档右下角浮动图标
- 悬停显示：常用快捷键提示
- 新手引导：首次使用时高亮重要功能

**快捷键体系**

## 5. 设计系统总结

### 5.1 飞书设计系统核心

#### 设计原则
1. **协作优先**：多人实时编辑，权限管理清晰
2. **效率至上**：快捷键丰富，Markdown 支持，模板库完善
3. **结构化表达**：块级编辑，层级清晰，自动生成目录
4. **跨设备一致**：统一字体避免渲染差异，响应式布局

#### 技术栈建议
- 布局系统：12 栅格系统
- 响应式断点：
  - Mobile: < 768px
  - Tablet: 768px - 1280px
  - Desktop: > 1280px
- 组件库：基于 React/Vue
- 状态管理：支持多人协作的 CRDT 算法

#### 关键特性
- **浮动工具栏**：减少固定 UI 占用空间
- **斜杠命令**：快速插入各类内容块
- **模板系统**：覆盖多种工作场景，提升用户粘性
- **权限体系**：细粒度权限控制，保障协作安全

### 5.2 Craft 设计系统核心

#### 设计原则
1. **极简美学**：纯净界面，专注内容本身
2. **块级编辑**：结构化组织，灵活嵌套

- **主题定制**：一键切换风格,支持自定义颜色和背景

### C. 色彩对比度标准（WCAG 2.0）

- 常规文本（<18pt）：至少 4.5:1

**Craft 色彩对比度检查**

## 总结

本文档详细分析了飞书和 Craft 两款应用的 UI/UX 设计方案，涵盖：

1. **页面布局结构**：从顶部导航、侧边栏、主内容区到响应式适配策略
2. **界面元素设计**：色彩系统、文字排版、UI 组件库的完整规范

4. **用户体验细节**：视觉层次、交互流程、可访问性的深度优化

**飞书**侧重**协作效率**，适合企业团队，强调实时多人编辑、权限管理和 Markdown 支持。

**Craft**侧重**内容美感**，适合个人创作者，强调极简设计、主题定制和块级编辑体验。

两者在浮动工具栏、块级编辑、快捷键系统等方面有共同的优秀实践，可根据产品定位选择性借鉴。
