# 🎨 如何查看 Craft 编辑器演示

## 快速开始

### 方法 1：从主界面访问（推荐）⭐

1. **启动开发服务器**（如果还没启动）
   ```bash
   npm run dev
   ```

2. **打开浏览器**
   访问：http://localhost:5177
   
3. **点击 "Craft Demo" 按钮**
   - 在主界面（Dashboard）的顶部
   - 找到橙红渐变色的 **"Craft Demo"** 按钮
   - 点击即可打开完整的 Craft 编辑器演示

4. **体验所有功能**
   - ✅ 选中文本 → 自动显示悬浮工具栏
   - ✅ 输入 `/` → 触发斜杠命令菜单
   - ✅ 悬停段落 → 显示拖拽手柄
   - ✅ 使用快捷键：Cmd+B (粗体)、Cmd+I (斜体)等

## 💡 使用提示

### 1. 悬浮工具栏
- 用鼠标选中任意文本
- 工具栏会自动出现在选中文本上方
- 支持：粗体、斜体、下划线、删除线、代码、标题、列表、链接

### 2. 斜杠菜单
- 在编辑器中输入 `/`
- 自动弹出命令菜单
- 可以搜索：输入关键词过滤
- 键盘导航：↑↓ 选择，Enter 确认

### 3. 块级操作
- 鼠标悬停在任意段落左侧
- 会出现拖拽手柄和添加按钮
- 可以拖拽重排序（功能开发中）

### 4. 快捷键
- `Cmd + B` - 粗体
- `Cmd + I` - 斜体
- `Cmd + U` - 下划线
- `/` - 打开命令菜单
- `Cmd + K` - 插入链接

## 🔧 如果遇到问题

### 服务器未启动
```bash
cd /Users/zhengzizhe/Documents/GitHub/Zkanso
npm run dev
```

### 看不到按钮
- 确保在 **Dashboard** 视图（不是编辑器视图）
- 刷新页面（Cmd + R）

### 样式不正确
- 检查是否引入了 CSS 文件
- 打开浏览器开发者工具查看控制台

### 功能不工作
- 打开浏览器控制台（F12）
- 查看是否有错误信息
- 确保所有依赖已安装：`npm install`
