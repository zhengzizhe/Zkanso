## 🎨 四、悬浮工具栏

### 4.1 何时出现

**触发条件**:
1. 用鼠标选中任意文本
2. 选中长度 > 0 个字符

**出现位置**:
```
       ┌──────────────────┐
       │ [B] [I] [U] ... │  ← 工具栏
       └──────────────────┘
            ↓
       [选中的文字]
```

- 在选中文字**上方** 50px
- 水平**居中**对齐选中区域
- 自动避免超出屏幕边缘

### 4.2 工具栏外观

**用户看到完整的工具栏**:
```
┌────────────────────────────────────────────────────────┐
│ [B] [I] [U] [S] [</>] │ [H1] [H2] [H3] │ [≡] [1.] ["] │ [🔗] [⋯] │
└────────────────────────────────────────────────────────┘
```

**容器样式**:
- 高度: 42px
- 内边距: 6px
- 圆角: 8px
- 背景: 半透明白色 `rgba(255, 255, 255, 0.98)`
- 边框: 1px 浅灰 `#EEEEEE`
- 毛玻璃效果: 背景模糊 12px

### 4.3 按钮详情

每个按钮的视觉:

| 按钮 | 图标 | 功能 | 快捷键 | 激活颜色 |
|------|------|------|--------|----------|
| **B** | 粗体 B | 加粗文字 | Cmd+B | 橙色 |
| **I** | 斜体 I | 斜体文字 | Cmd+I | 橙色 |
| **U** | 下划线 U | 下划线 | Cmd+U | 橙色 |
| **S** | 删除线 S | 删除线 | - | 橙色 |
| **</>** | 代码 | 行内代码 | Cmd+E | 橙色 |
| **\|** | 分隔符 | 视觉分组 | - | 灰色 |
| **H1** | 标题1 | 一级标题 | - | 橙色 |
| **H2** | 标题2 | 二级标题 | - | 橙色 |
| **H3** | 标题3 | 三级标题 | - | 橙色 |
| **≡** | 列表 | 无序列表 | - | 橙色 |
| **1.** | 列表 | 有序列表 | - | 橙色 |
| **"** | 引用 | 引用块 | - | 橙色 |
| **🔗** | 链接 | 插入链接 | Cmd+K | 橙色 |
| **⋯** | 更多 | 更多选项 | - | 橙色 |

**按钮规格**:
- 尺寸: 30px × 30px
- 圆角: 6px
- 图标: 16px
- 间距: 4px

**默认状态**:
- 背景: 透明
- 图标颜色: 灰色 `#616161`

**悬停状态**:
- 背景: 浅灰 `rgba(0, 0, 0, 0.04)`
- 图标颜色: 深黑 `#212121`
- 过渡: 100ms

**激活状态** (格式已应用):
- 背景: Craft 橙 `#FF6C47`
- 图标颜色: 白色 `#FFFFFF`
- 阴影: 轻微阴影

**点击动画**:
- 缩小到 95%
- 持续 100ms

### 4.4 出现/消失动画

**出现动画** (150ms):
1. 透明度: 0 → 1
2. 位移: 向下 8px → 0
3. 缩放: 95% → 100%

**消失动画** (150ms):
1. 透明度: 1 → 0

3. 缩放: 100% → 95%
